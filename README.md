# RenderVerse AI

RenderVerse AI adalah aplikasi berbasis Streamlit yang mengubah screenshot SketchUp menjadi render photorealistic menggunakan AI dari Replicate (Flux Pro).

## Fitur

- Upload screenshot SketchUp
- Mengubah model menjadi render photorealistic
- Pengaturan pencahayaan
- Pengaturan material
- Pengaturan environment
- Pengaturan kamera

## Teknologi yang digunakan

- Python
- Streamlit
- Replicate API
- Flux Pro AI Model
- Pillow
